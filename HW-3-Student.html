Let \(E[X] = \mu\). Show that \(Var[X] := E[(X-E[X])^2] = E[X^2]-(E[X])^2\). Note, all you have to do is show the second equality (the first is our definition from class).

Expand \(E[(X-E[X])^2]\) into \(E[(X^2 - 2X*E[X] + E[X]^2)]\). Now, apply linearity of expectation to get \(E[X^2] - E[2X*E[X]] + E[E[X]^2]\). Since E[X] is a constant, and the expected value of a constant is the constant, simplify to \(E[X^2] - (2E[X] * E[X]) + E[X]^2 = E[X^2] - 2E[X]^2 + E[X]^2\). Finally, this simplifies to \(E[X^2] - E[x]^2\).

Discuss the trade-off in accuracy and interpretability in pruning the above tree.

Student Input

The benefit of pruning a decision tree is that it simplifies the tree down which helps with avoiding overfitting, so it can be generalized. Pruning also makes a decision tree more interpretable because there will be less splits. The downsides of pruning is that you lose perfect purity. The collapsed branches will not have perfectly pure nodes.

3.6

Discuss the ways a decision tree could manifest algorithmic bias.

Student Answer

There are 3 main ways a decision tree could manifest algorithmic bias, with are Independence, Separation and Sufficiency. Independence looks at whether the decision tree has desperate impact or statistical parity, and considers a tree biased if it fulfills either of those conditions. Separation checks if the prediction is independent of the protected variable conditional to the ground truth, and looks at if it violates equilized odds. Finally, Sufficiency checks if the outcome of tree is independent from the groups they came from.
